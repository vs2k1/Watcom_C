 Библиотека для работы с числами прописью

 CopyRight 1997 Vladimir STUkan                              
 ************************************************************
 RUBLI_B - число пpописью с большой буквы
 RUBLI_S - число пpописью с маленькой буквы
 RUBLI_R - число прописью с большой буквы + "рубль/рублей" 
 RUBLI_K - число прописью с маленькой буквы + "копейка/копеек"
 KOP     - копейки цифрой и прописью (пример: 21 копейка)
 ------------------------------------------------------------
 Использование:

 SET LIBRARY TO RUBLI ADDITIVE

 RUBLI_B("Текст",число)

 Пpимеp :

 STORE 123 TO Var
 ?RUBLI_B("РУБЛИ : ",Var)
 ?RUBLI_B("ДОЛЛАРОВ США : ",200)
 ?RUBLI_R("",12345)
 
 ********************
 STORE 123.44 TO Cifr
 RUB_KOPa(Cifr)
 RUB_KOPb(Cifr)

 ********************
 FUNCTION   RUB_KOPa
 ********************
 PARAMETER Var
 RETURN RUBLI_R("",Var)+RUBLI_K("",INT(Var*100)-INT(Var)*100)
 
 ********************
 FUNCTION   RUB_KOPb
 ********************
 PARAMETER Var
 RETURN RUBLI_R("",Var)+KOP("",INT(Var*100)-INT(Var)*100)
